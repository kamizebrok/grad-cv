# Portfolio CV 

My interactive CV/portfolio built with **React + Vite** and deployed on GitHub Pages.

## 🔧 Technologies Used
- React 18
- Vite
- GitHub Pages (hosting)

## 🎯 Features
- Responsive design
- Clean, component-based architecture
- Easy to update and maintain
